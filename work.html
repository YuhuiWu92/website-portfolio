Client My Flix

Client side application created using React and JavaScript. Interacts with server side REST API and MongoDB database. Users can register new accounts, login using existing accounts, and update and delete their current account. Renders a list of movies stored in a MongoDB database. Users can view detailed information about each movie and add and remove movies from their favorite list.
